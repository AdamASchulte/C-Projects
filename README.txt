Summary: Singly Linked List in C. Employs data structures, memory management, and user IO.
         It allows users to add nodes to the end of the List, add nodes to a specified location
         within the List, remove a node at a specified location within the list, remove the
         first instance of a node with a specified value, or view the value of a node at a 
         specific index of the List. (0 Indexed)
         
To Compile: gcc -o LinkedList.exe SinglyLinkedList.c 
To Run: .\LinkedList.exe
